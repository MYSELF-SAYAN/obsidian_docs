# Note Relationships

Backlinks and link management services for Obsidian.

## Services

| Service | Description |
|---------|-------------|
| [Backlinks](./relationships/backlinks) | View and manage incoming links |
| [Links](./relationships/links) | Internal link operations |

## Overview

The Note Relationships group provides services for understanding and managing the connections between notes in your vault.

### Key Capabilities

- **Backlinks**: Track what links to a specific note
- **Unlinked Mentions**: Find references to a note that aren't explicitly linked
- **Link Resolution**: Resolve and validate internal links
- **Graph Data**: Access graph-view relevant data

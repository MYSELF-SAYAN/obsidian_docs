## Overview

The Files & Vault group contains the essential services for managing files, folders, and vault-level operations in Obsidian. These are the foundational services most users will interact with regularly.

### Key Capabilities

- **File Operations**: Read, write, create, delete, and modify files
- **Vault Management**: Configure and control vault settings
- **Template Support**: Work with note templates
- **Search**: Full-text search across your vault
- **Navigation**: File tree browsing and outline navigation

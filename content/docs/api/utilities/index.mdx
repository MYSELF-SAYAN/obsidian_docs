System utilities and developer tools for Obsidian.

## Overview

The System & Utilities group provides services for system-level operations, debugging, and general utilities.

### Key Capabilities

- **File Recovery**: View and restore previous versions
- **Version Comparison**: See what changed between versions
- **Developer Tools**: Debug mode, DOM inspection, screenshots
- **App Control**: Reload, restart, version info
- **Statistics**: Word count and document statistics

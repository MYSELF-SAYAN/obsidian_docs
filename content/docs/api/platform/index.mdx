# Platform Configuration

Obsidian platform configuration and plugin services.

## Services

| Service | Description |
|---------|-------------|
| [Plugins](./platform/plugins) | Plugin management |
| [Themes](./platform/themes) | Theme installation and management |
| [Snippets](./platform/snippets) | CSS snippet management |

| [Publish](./platform/publish) | Obsidian Publish site management |

## Overview

The Platform Configuration group contains services for managing Obsidian's appearance, plugins, and cloud services.

### Key Capabilities

- **Plugin Management**: Install, enable, disable plugins
- **Theme Customization**: Switch and manage visual themes
- **CSS Snippets**: Custom styling modifications
- **Cloud Sync**: Vault synchronization across devices
- **Publishing**: Share notes publicly via Obsidian Publish

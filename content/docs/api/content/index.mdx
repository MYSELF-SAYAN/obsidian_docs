# Content & Metadata

Note content and metadata management services for Obsidian.

## Services

| Service | Description |
|---------|-------------|
| [Properties](./content/properties) | Frontmatter and note properties |
| [Aliases](./content/aliases) | Note alias management |
| [Tasks](./content/tasks) | Task and todo management |
| [Random](./content/random) | Random note selection |

## Overview

The Content & Metadata group provides services for managing the content, properties, and metadata of your notes.

### Key Capabilities

- **Frontmatter**: Read and write YAML frontmatter
- **Aliases**: Manage note alternative names
- **Tasks**: Create, complete, and track tasks
- **Random Notes**: Get random notes for spaced repetition or review

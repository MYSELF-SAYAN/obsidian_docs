# Obsidian SDK Documentation

Welcome to the **Obsidian SDK** documentation. This SDK provides a comprehensive, type-safe interface to Obsidian's CLI, enabling you to programmatically manage vaults, notes, files, search, linking, and more.

## Modules

| Module | Description |
| ------ | ----------- |
| [Quick Start](/docs/quickstart) | Get up and running in 2 minutes. |
| [Notes](/docs/notes) | Full CRUD for markdown notes. |
| [Files](/docs/files) | Enumerate and filter vault files. |
| [Daily](/docs/daily) | Daily notes and task management. |
| [Linking](/docs/linking) | Backlinks, orphans, graph queries. |
| [Search](/docs/search) | Full-text and contextual search. |
| [Vaults](/docs/vaults) | List and switch vaults. |
| [FileOps](/docs/fileOps) | Move, copy, trash file operations. |
| [Templates](/docs/templates) | Insert and manage templates. |
| [Plugins](/docs/plugins) | Enable, disable, query plugins. |
| [Dev](/docs/dev) | Developer/debug utilities. |
| [History](/docs/history) | File version history. |
| [Workspace](/docs/workspace) | Workspace layout management. |
| [Sync](/docs/sync) | Obsidian Sync integration. |
| [Random](/docs/random) | Random note utilities. |
| [Metrics](/docs/metrics) | Vault metrics and stats. |
| [SystemInfo](/docs/systemInfo) | System and environment info. |

Each module follows the same pattern: instantiate via the `Obsidian` class and call async methods.

```ts
import { Obsidian } from "obsidian-sdk";

const obsidian = new Obsidian({ vault: "myVault" });
const files = await obsidian.files.list();
```

For the full CLI reference, visit the [Obsidian CLI docs](https://obsidian.md/help/cli).
